Grids
=====

The following predefined grid models are available. The models can be called by their names, given in parentheses.

.. _weak-lv-grid:

**Weak low-voltage grid** (``Weak_LV_Grid``)

   :widths: 30 70
   :header-rows: 1

   * - Parameter
     - Value 
   * - :math:`V_{g,R}`
     - :math:`400\,\mathrm{V}`
   * - :math:`I_{g,R}`
     - :math:`18\,\mathrm{A}`
   * - :math:`f_{g,R}`
     - :math:`50\,\mathrm{Hz}`
   * - :math:`R_g`
     - :math:`0.07\,\Omega`
   * - :math:`L_g`
     - :math:`30\,\mathrm{mH}`
