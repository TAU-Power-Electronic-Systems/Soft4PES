Filters
=======

.. _lcl-filter-1300:

The following predefined filter models are available. The models can be called by their names, given in parentheses.


:math:`\mathrm{LCL}` **filter** (:math:`f_r = 1300\,\mathrm{Hz}`) (``LCL_Filter_fr_1300``)

.. list-table::
   :widths: 30 70
   :header-rows: 1

   * - Parameter
     - Value
   * - :math:`L_{\mathrm{fc}}`
     - :math:`3\,\mathrm{mH}`
   * - :math:`R_{\mathrm{fc}}`
     - :math:`0.1\,\Omega`
   * - :math:`C`
     - :math:`10\,\mu\mathrm{F}`
   * - :math:`R_c`
     - :math:`1\,\mathrm{m}\Omega`
   * - :math:`L_{\mathrm{fg}}`
     - :math:`3\,\mathrm{mH}`
   * - :math:`R_{\mathrm{fg}}`
     - :math:`0.1\,\Omega`
