Converters
==========

.. _2l-lv-converter:

The following predefined converter models are available. The models can be called by their names, given in parentheses.

**2-Level low-voltage converter** (``2L_LV_Converter``)

.. list-table::
   :widths: 30 70
   :header-rows: 1

   * - Parameter
     - Value
   * - :math:`V_{\mathrm{dc}}`
     - :math:`750\,\mathrm{V}`
   * - :math:`n_{\mathrm{conv}}`
     - :math:`2`
