Grid Systems
============

.. _grid-systems:

The following predefined grid systems are available. The systems can be called by their names, given in parentheses.

**2-level converter connected to a weak low-voltage grid via an** :math:`\mathrm{LCL}` **filter**

Grid: :ref:`Weak low-voltage grid <weak-lv-grid>`

Filter: :math:`\mathrm{LCL}` :ref:`filter <lcl-filter-1300>` (:math:`f_r = 1300\,\mathrm{Hz}`)

Converter: :ref:`2-Level converter <2l-lv-converter>`


**2-level converter connected to a strong low-voltage grid via an** :math:`\mathrm{LCL}` **filter**

Grid: :ref:`Strong low-voltage grid <strong-lv-grid>`

Filter: :math:`\mathrm{LCL}` :ref:`filter <lcl-filter-1300>` (:math:`f_r = 1300\,\mathrm{Hz}`)

Converter: :ref:`2-Level converter <2l-lv-converter>`


**3-level converter connected to a strong medium-voltage grid without a filter**

Grid: :ref:`Strong medium-voltage grid <strong-mv-grid>`

Filter: :ref:`No filter <no-filter>`

Converter: :ref:`3-Level converter <3l-mv-converter>`


**3-level converter connected to a strong medium-voltage grid via an** :math:`\mathrm{LCL}` **filter**

Grid: :ref:`Strong medium-voltage grid <strong-mv-grid>`

Filter: :math:`\mathrm{LCL}` :ref:`filter <lcl-filter-380>` (:math:`f_r = 380\,\mathrm{Hz}`)

Converter: :ref:`3-Level converter <3l-mv-converter>`
