class soft4pes.model.machine.PMSM(par, conv, base, T_ref_init, mtpa_lut)[source]#

Bases: soft4pes.model.common.system_model.SystemModel

Permanent magnet synchronous machine (PMSM) model. The model operates at a constant electrical angular rotor speed. The system is modelled in a dq-frame, where the d-axis is aligned with the rotor flux. However, the state of the system is the stator current in the alpha-beta frame, and thus reference frame conversions are performed during state updates.

Parameters:

  • par (PMSMParameters) – Permanent magnet synchronous machine parameters in p.u.

  • conv (converter object) – Converter object.

  • base (base value object) – Base values.

  • T_ref_init (float) – Initial torque reference [p.u.].

  • i_mag_points (int, optional) – Number of current magnitude points for MTPA trajectory generation. The default is 101.

  • theta_points (int, optional) – Number of angle points for MTPA trajectory generation. The default is 2001.
