class soft4pes.model.grid.base_grid.BaseGrid(Vgr, Igr, fgr)[source]#

Base values for a grid.

The class computes the base values for a grid based on the rated values.

Parameters:

  • Vgr (float) – Rated voltage [V] (line-to-line rms voltage).

  • Igr (float) – Rated current [A] (line rms current).

  • fgr (float) – Rated frequency [Hz].
