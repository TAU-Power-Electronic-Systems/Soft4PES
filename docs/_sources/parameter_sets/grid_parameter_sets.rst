Grid Parameter Sets
===================

The examples can be used with user defined parameters, or with predefined components and systems. 

**Grid Systems** are combinations of individual components, such as grids, filters, and converters. **Grid Components** are the building blocks of these systems, allowing users to create their own systems by combining different components.
